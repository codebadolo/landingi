Politique de Confidentialité

Chez Ibaara, la protection de vos données personnelles est une priorité. Cette politique explique quelles informations nous collectons, pourquoi nous les collectons et comment nous les protégeons.

1. Informations collectées

Nous collectons uniquement les informations nécessaires pour vous offrir une expérience optimale : nom, numéro de téléphone, localisation approximative, types de services recherchés ou proposés.

2. Utilisation des données

Les données collectées servent principalement à :

  • Mettre en relation les clients avec les prestataires.
  • Améliorer la qualité du service.
  • Personnaliser les recommandations.
  • Garantir la sécurité des comptes.

3. Partage des données

Vos données ne sont jamais vendues. Elles peuvent être partagées uniquement avec :

  • Des partenaires de confiance respectant le RGPD.
  • Les autorités si la loi l’exige.

4. Sécurité

Nous utilisons des technologies modernes pour protéger vos données : chiffrement, audit régulier, anonymisation partielle lorsque possible.

5. Vos droits

Conformément aux lois en vigueur, vous pouvez :

  • Demander l’accès à vos données.
  • Demander leur suppression.
  • Modifier vos informations.
  • Retirer votre consentement à tout moment.

6. Contact

Pour toute question concernant votre confidentialité, contactez-nous à : support@ibaara.com
